\section{Executive Summary}

PlayScore is a professional bilingual game review platform built with Laravel 11, featuring AI-powered recommendations, user engagement through comments, and comprehensive game management capabilities. The platform serves 14 curated game reviews with advanced features including real-time AI chatbot integration, skeleton loading for enhanced UX, and bilingual support (English/Arabic).

\subsection{Key Features}
\begin{itemize}[noitemsep]
    \item Complete CRUD operations for game reviews with admin dashboard
    \item User authentication and role-based authorization
    \item AI chatbot using Google Gemini for personalized recommendations
    \item User commenting system with authorization controls
    \item RAWG API integration for automatic game metadata fetching
    \item Bilingual interface (English/Arabic) with 60+ translation keys
    \item Server-side caching (5min lists, 1hr API) and skeleton loading
    \item Responsive dark theme design
\end{itemize}

\section{System Architecture}

\subsection{Technology Stack}

\textbf{Backend:} Laravel 11 (PHP 8.2+) provides MVC architecture, Eloquent ORM, built-in authentication, and caching capabilities for rapid, secure development.

\textbf{Database:} MySQL 8.4 handles relational data with three main tables: users, games, and comments, utilizing foreign keys and cascading deletes.

\textbf{Frontend:} Bootstrap 5 with Vanilla JavaScript ensures responsive design and optimal performance without unnecessary dependencies.

\textbf{External APIs:}
\begin{itemize}[noitemsep]
    \item \textit{RAWG API:} Comprehensive game database for metadata and images
    \item \textit{Google Gemini AI:} Natural language processing for chatbot recommendations
\end{itemize}

\subsection{System Components}

\textbf{Controllers:}
\begin{enumerate}[noitemsep]
    \item GameController: CRUD operations, RAWG API integration, caching
    \item ChatController: AI chatbot with conversation context
    \item AuthController: Authentication and session management
    \item CommentController: User comments with authorization
    \item LocaleController: Language switching
\end{enumerate}

\textbf{Models:} User, Game, Comment with Eloquent relationships and cascading deletes

\textbf{Views:} Blade templates including layouts, authentication pages, admin dashboard, and game pages

\section{Key Implementations}

\subsection{Automation Features}
\begin{itemize}[noitemsep]
    \item \textbf{Metadata Fetching:} Automatic retrieval of game title, description, release date, and images from RAWG API
    \item \textbf{Caching Strategy:} 5-minute cache for game lists, 1-hour cache for RAWG responses
    \item \textbf{Cache Invalidation:} Automatic flush on create/update/delete operations
    \item \textbf{Lazy Loading:} Images load on-demand for improved performance
    \item \textbf{Skeleton Loading:} Instant visual feedback during data fetching
\end{itemize}

\subsection{API Endpoints}

\textbf{Public Endpoints:}
\begin{itemize}[noitemsep]
    \item \texttt{GET /api/games/popular} - Paginated popular games
    \item \texttt{GET /api/games/recent} - Recent reviews
    \item \texttt{GET /api/games/search?q=\{query\}} - Search functionality
    \item \texttt{GET /api/games/\{gameId\}/comments} - Fetch comments
\end{itemize}

\textbf{Authenticated Endpoints:}
\begin{itemize}[noitemsep]
    \item \texttt{POST /api/games/\{gameId\}/comments} - Post comment
    \item \texttt{DELETE /api/games/\{gameId\}/comments/\{id\}} - Delete comment
    \item \texttt{POST /api/chat} - AI chatbot interaction
\end{itemize}

\textbf{Admin Endpoints:}
\begin{itemize}[noitemsep]
    \item \texttt{POST /api/games} - Create review
    \item \texttt{PUT /api/games/\{id\}} - Update review
    \item \texttt{DELETE /api/games/\{id\}} - Delete review
\end{itemize}

\section{Technical Challenges \& Solutions}

\subsection{Major Challenges Resolved}

\begin{enumerate}
    \item \textbf{RAWG API Performance}
    \begin{itemize}[noitemsep]
        \item Problem: API rate limiting and slow responses
        \item Solution: 1-hour caching with 5-second timeout
    \end{itemize}
    
    \item \textbf{Conversation Context}
    \begin{itemize}[noitemsep]
        \item Problem: AI chatbot lacked memory
        \item Solution: Full conversation history sent with each request
    \end{itemize}
    
    \item \textbf{User Experience}
    \begin{itemize}[noitemsep]
        \item Problem: Blank screens during loading
        \item Solution: Skeleton placeholders with animated shimmer
    \end{itemize}
    
    \item \textbf{Authorization}
    \begin{itemize}[noitemsep]
        \item Problem: Comment deletion permissions
        \item Solution: Authorization check allowing owner or admin deletion only
    \end{itemize}
    
    \item \textbf{Internationalization}
    \begin{itemize}[noitemsep]
        \item Problem: JavaScript content not translated
        \item Solution: Exposed translations via window.translations object
    \end{itemize}
\end{enumerate}
